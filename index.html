Enable Chrome GLIC in CN

中国地区一键激活 Google Chrome 浏览器全新 Gemini AI 侧边栏 (GLIC) 功能

技术原理 (Technical Principles)

Google Chrome 的 AI 侧边栏功能 (GLIC) 在部分地区被默认禁用。本项目通过修改 Chrome 的全局配置文件 Local State 中的以下字段，强制激活 GLIC 资格：

字段 (JSON Key) 修改值 (Value) 作用说明 (Description)
variations_country "us" 强制设置浏览器区域为美国 (US)
variations_permanent_consistency_country ["1", "us"] 永久一致性地区设置为美国 (US)
is_glic_eligible true 对所有用户配置文件启用 GLIC 体验资格

使用指南 (Usage Guides)

  1. 下载/准备脚本

    进入本仓库的 scripts/ 目录，拷贝或下载以下两个文件到同一个目录下：

    下载 win_enable_glic.bat (引导器) 下载 win_enable_glic.ps1 (核心脚本)

  2. 一键运行

    鼠标右键点击 win_enable_glic.bat，选择 “以管理员身份运行” (或者直接双击运行)。

  3. 完成修改

    脚本会自动为您强行关闭正在运行的 Chrome 进程，并在自动备份原配置后完成修改。提示成功后重新打开 Chrome 即可。

关于安全性与备份

修改前，脚本会自动在 Chrome 配置文件夹中生成一个 Local State.backup.时间戳 格式的备份文件。如遇任何异常，可随时恢复备份。

  1. 准备脚本文件

    确保您已将本仓库的 scripts/ 文件夹下载到本地，或直接下载 macOS 专属脚本：

    下载 mac_enable_glic.sh (引导器) 下载 mac_enable_glic.py (核心脚本)

  2. 终端执行

    打开 Mac 的“终端 (Terminal)”，切换到脚本所在目录，执行以下命令：

    chmod +x mac_enable_glic.sh && ./mac_enable_glic.sh

  3. 重启 Chrome

    脚本运行完成后，重新启动 Chrome 浏览器，点击侧边栏即可享用 GLIC 功能。
